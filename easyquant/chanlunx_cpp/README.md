### 如何编译

#### 前提条件

- Visual Studio 2015 或者 Visual Studio 2017 或者只要 Windows Build Tools也是可以编译的，其它编译工具没有测试但是也应该可以编译。
- 安装CMake。

### 编译命令

在项目的根目录下依次执行以下命令。

```
mkdir build
cd build
cmake ..
cmake --build . --config Release
```

或者用下面命令生成Debug版本。

```
cmake --build . --config Debug
```
